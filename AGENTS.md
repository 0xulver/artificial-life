# Agent Rules

## Commit Guidelines

When creating git commits in this repository:

1. **Do NOT include** co-author trailers like:
   - `Co-authored-by: Sisyphus <clio-agent@sisyphuslabs.ai>`
   - Any AI/agent co-author attribution

2. **Do NOT include** footer messages like:
   - `Ultraworked with [Sisyphus](https://github.com/code-yeongyu/oh-my-opencode)`
   - Any promotional or attribution links

3. **Keep commits clean** - only include the commit message and description of changes.
